Handwriting Recognition Application

This application allows users to upload a single handwritten character, letters and numbers included. The image will then be processsed and analysed by our computational model. Our system will then attempt to correctly predict the character. We have trained our model with the use of the NIST data set which contains 810,000 images of which we will use a portion of. This data set contains handwritten characters from 3600 writers, of numbers and letters, uppercase and lowercase included.
